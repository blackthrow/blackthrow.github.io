Blackthrow

, a small computer that can be hidden inside government agencies or corporations. It connects to the Tor or I2P networks and publishes its SSH server as a hidden service in any of these networks. A freedom fighter can then connect to the blackthrow anonymously and remote control it to deliver any type of packets to any location at the internets, that the host organization can connect to.

— Telecomix

This is an experiment.

To design an operating system for Telecomix's blackthrow computer.

A Raspberry Pi as the platform – affordable, abundant and with just enough computing power.

Controlled via SSH server, the server is reachable as a hidden service. All sessions are forced into isolation inside a container.

Conducted as a community effort, distributed as a free software.
